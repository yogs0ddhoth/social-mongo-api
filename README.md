# social-mongo-api

A functional API that uses Mongoose to query a [MongoDB](https://www.mongodb.com/) database. Documents are returned as JSON objects.

## Dependencies
* [Express](https://expressjs.com/)
* [Mongoose](https://mongoosejs.com/)
